# CTF Writeups

A personal collection of writeups for CTF challenges I've solved across various platforms.

## Status

This repo is a work in progress. I'm in the process of documenting and uploading all the challenges I've completed, each with a full walkthrough and explanation of my thought process.

## Platforms

- **Hack The Box (HTB)**
- **TryHackMe**
- **PicoCTF**

## What to Expect

Each writeup will cover the challenge objective, the tools and techniques used, and a step-by-step solution.

## Disclaimer

All challenges documented here are from legal, sanctioned CTF platforms. Writeups are only published for retired or completed challenges where sharing solutions is permitted.
